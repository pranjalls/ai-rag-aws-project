# AI RAG on AWS

Project: AI-Powered Document Q&A System on AWS
Tech: Amazon Bedrock, Textract, OpenSearch, Lambda, Terraform, GitHub Actions, DevSecOps

Built an AI-powered RAG-based document question-answering system using AWS services. Implemented serverless architecture with Lambda and API Gateway, vector search using OpenSearch, GenAI using Amazon Bedrock, automated CI/CD with GitHub Actions, and secured the system using DevSecOps best practices.
